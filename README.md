## Project Description

This project aims to develop a Digital Prescription & Smart Medication Delivery Management System that enables doctors to create digital prescriptions, pharmacies to manage medication orders, and delivery personnel to track and deliver medications efficiently.

## Features

- Digital prescription management
- Patient management
- Medication inventory tracking
- Order processing
- Delivery tracking
- Secure user authentication

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- CSV File Storage

## Design Patterns

- Factory Method Pattern
- Observer Pattern
- Facade Pattern
